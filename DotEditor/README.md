# ドット絵エディタ

ブラウザで動作する単一HTMLファイルのドット絵エディタです。インストール不要。

---

## 起動方法

`Dot_Editor.html` をブラウザで開くだけ。

---

## 画面構成

```
[ メニューバー ]
[ ツールバー   ]
┌─────────┬──────────────────┬──────────┐
│ 左パネル  │   キャンバス      │ 右パネル  │
│ カラー   │                  │ナビゲータ │
│ パレット  │                  │ レイヤー  │
│ツールオプ │                  │          │
├─────────┴──────────────────┴──────────┤
│ フレームバー（アニメーション）            │
├───────────────────────────────────────┤
│ ステータスバー                          │
└───────────────────────────────────────┘
```

---

## メニューバー

| ボタン | 機能 |
|---|---|
| 新規 | キャンバスを新規作成（W/H入力欄のサイズで作成） |
| リサイズ | キャンバスサイズを変更 |
| PNG書出 | 全レイヤーを統合してPNGで保存 |
| GIF書出 | アニメーションをGIFで書き出し |
| 💾 保存 | プロジェクトを `.dotpix` (JSON) で保存 |
| 📂 読込 | `.dotpix` / `.json` プロジェクトを読み込み |
| コピー | 選択範囲（なければ全体）をコピー |
| 切り取り | 選択範囲を切り取り |
| 貼り付け | クリップボードから貼り付け |
| W / H | 新規作成時のキャンバスサイズ指定 |
| カラーモード | フルカラー / 65535色 / 256色パレット / 16色パレット |
| ⚙ 設定 | マウスボタンの割り当てを変更 |

## カラー（左パネル）

- **描画色ボックス**（前面）: クリックでカラーピッカーを開く
- **背景色ボックス**（背面）: クリックでカラーピッカーを開く
- **⇄ ボタン**: 描画色と背景色を入れ替え
- **HEXテキストボックス**: 直接 `#RRGGBB` で入力可能
- **Alt + クリック（キャンバス上）**: スポイト

### マウスボタン割り当て（⚙ 設定から変更可）
| ボタン | デフォルト動作 |
|---|---|
| 左 | 描画（描画色） |
| 中 | 消しゴム |
| 右 | 描画（背景色） |

---

## パレット（左パネル）

256色・16色パレットモード時に表示されます。

### 基本操作
| 操作 | 内容 |
|---|---|
| 左クリック | 描画色にセット |
| 右クリック | 背景色にセット |
| ダブルクリック | カラーピッカーで色を編集 |
| D&D（単セル） | 2つのセルの色をスワップ |

### ハイライト表示
| 表示 | 意味 |
|---|---|
| 白枠 | 現在の選択範囲 |
| 薄い青枠 | キャンバス上のマウス位置の色に対応するセル |
| オレンジ枠 | 差し替えモードで選んだ入替先 |

### 範囲選択
- パレット上で**左ボタンドラッグ**することで複数セルを選択（白枠）
- 選択すると操作パネルが表示される

### 選択範囲の操作パネル

| ボタン | 機能 |
|---|---|
| グラデ | 選択範囲の両端2色の間を線形補間してグラデーションを生成 |
| 反転 | 選択範囲の全色をRGB反転（255-R, 255-G, 255-B） |
| 差し替え | 押した後に別の範囲をドラッグ選択すると2つの範囲の内容をスワップ |
| ✕解除 | 選択を解除 |

### 範囲の移動
選択範囲内から**中ボタンドラッグ**すると、選択範囲をパレット内の別位置へ移動（他の色はシフトされる）。

### 追従 ON / OFF
- **ON（デフォルト・緑）**: パレット操作時、キャンバス上の該当ピクセルの色も同時に更新される
- **OFF**: パレット表示のみ変更、描かれたピクセルはそのまま

### パレットの保存・読込
- 💾 でJSON形式で保存
- 📂 でJSON形式から読み込み

## レイヤー（右パネル）

| ボタン | 機能 |
|---|---|
| ＋ | レイヤー追加 |
| － | アクティブレイヤーを削除 |
| ⧉ | アクティブレイヤーを複製 |
| ↧ | 1つ下のレイヤーに結合 |
| 👁 | 表示・非表示切替 |
| 🔒 / 🔓 | ロック・アンロック（ロック中は編集不可） |
| 🎭 | マスクがある場合に表示 |

レイヤー名をダブルクリックで名前変更。

### レイヤー設定
- **合成モード**: 通常 / 乗算 / スクリーン / 加算
- **不透明度**: 0〜100%

### マスク
- **追加**: 白いマスクを作成（白=不透明、黒=透明）
- **編集**: マスクを直接描画
- **適用**: マスクをレイヤーに焼き込んで削除

## アニメーション（フレームバー）

| 操作 | 機能 |
|---|---|
| ＋ | フレーム追加 |
| － | 現在フレームを削除 |
| ⧉ | 現在フレームを複製 |
| ▶ 再生 | アニメーション再生・停止 |
| フレームサムネイル | クリックでそのフレームを選択 |
| 数値入力（ms/frame） | そのフレームの表示時間（ミリ秒）を指定 |

各フレームは独立したレイヤー構成を持ちます。

## カラーモード

| モード | 説明 |
|---|---|
| フルカラー | 制限なし（任意のRGB） |
| 65535色 | R:3bit / G:3bit / B:3bit 相当に丸め |
| 256色パレット | 216色(6×6×6)+24グレースケールのパレット。カスタマイズ可 |
| 16色パレット | 標準16色固定パレット。カスタマイズ可 |

モード切替時は既存のピクセルが新しいパレットに自動変換されます。

---

## プロジェクトファイル形式 (.dotpix)

JSON形式。以下を保存：
- キャンバスサイズ・カラーモード
- 全フレーム・全レイヤー（PNG base64）
- パレット
- アニメーションフレームの遅延時間

---

## 動作環境

モダンブラウザ（Chrome / Firefox / Edge / Safari）でローカルファイルとして動作します。サーバー不要。
