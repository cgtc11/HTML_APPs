# PaintStudio

ブラウザで動作するシングルファイルのペイントアプリケーションです。  
インストール不要。HTMLファイルを開くだけで使えます。

---

## 使い方

`PaintStudio.html` をブラウザで開くだけです。サーバー不要、インターネット接続不要。

---

## ブラシ一覧

| ブラシ | ショートカット | 説明 |
|--------|:-----------:|------|
| インクペン | B → 種類選択 | 完全鮮明なストローク。主線・線画向け |
| ソフト円 | B → 種類選択 | ぼかし付きエアブラシ。硬さで拡散量を調整 |
| ハード円 | B → 種類選択 | 硬さ100でシャープ、下げるとソフトエッジ |
| マーカー | B → 種類選択 | 完全不透明のフラット塗り |
| 鉛筆 | B → 種類選択 | ノイズテクスチャで手描き感 |
| カリグラフィ | B → 種類選択 | 45度楕円の筆圧変化風ストローク |
| 水彩 | B → 種類選択 | にじみ＋エッジ色だまり。重ね塗りで濃くなる |
| 平筆 | B → 種類選択 | 角型先端のスクエアストローク |
| ドットペン | D | ピクセル単位の矩形ドット |
| 消しゴム | E | 透明に戻す |
| ぼかし | U | ドラッグで周辺をぼかす |

## ツール一覧

| ツール | ショートカット | 説明 |
|--------|:-----------:|------|
| ブラシ | `B` | 上記ブラシ種類で描画 |
| ドットペン | `D` | ピクセルドット描画 |
| 消しゴム | `E` | 透明化消去 |
| ぼかし | `U` | 領域ぼかし |
| 塗りつぶし | `G` | 許容値20でフラッドフィル |
| 矩形選択 | `S` | 矩形範囲を選択。ハンドルでリサイズ可 |
| 自由選択 | `A` | フリーハンドでなぞった範囲を選択 |
| 自動選択 | `W` | 色が近いピクセルを一括選択（許容値・隣接・全レイヤー設定あり） |
| 移動 | `V` | 選択範囲またはレイヤー全体を移動 |
| ライン | `L` | 直線を描画 |
| 四角形 | `R` | 矩形を描画（塗り・枠線選択可） |
| 楕円 | `O` | 楕円を描画（塗り・枠線選択可） |
| テキスト | `T` | テキストボックスで文字を配置（横書き・縦書き対応） |
| スポイト | `Alt` 押し | 任意の位置から色を取得 |

## マウス操作

| 操作 | デフォルト動作 |
|------|--------------|
| 左クリック・ドラッグ | 描画色で描く |
| 右クリック・ドラッグ | 背景色で描く |
| 中クリック・ドラッグ | パン移動 |
| `Ctrl` ＋ ホイール | ズーム |
| `Space` ＋ ドラッグ | パン移動 |
| `Alt` ＋ クリック | スポイト（色取得） |

> 🖱 **マウスボタン設定** ボタンから、各ボタンの割り当てとホイールの動作（ズーム / ブラシサイズ）を変更できます。

## レイヤー

- **追加 / 削除 / 上下移動** をパネル右上ボタンで操作
- **レイヤー名** をダブルクリックで変更
- **不透明度** を数値入力で変更（0〜100）
- **ブレンドモード**: 通常 / 加算 / スクリーン / 乗算
- **下のレイヤーと統合** 可能
- **目玉アイコン** で表示・非表示切り替え

> Undo はレイヤーごとに最大 **50ステップ** 保持します。

---

## カラーピッカー

- **SV（彩度・明度）フィールド** をドラッグで色選択
- **色相バー** でベースカラーを変更
- **アルファバー** で透明度を調整
- **HEX入力**（例: `#ff4466`）
- **R / G / B / A** を数値で直接入力
- **描画色 / 背景色** を切り替えて両方設定可能
- **⇄ ボタン** で描画色と背景色を入れ替え

---

## 選択範囲の操作

| 操作 | 方法 |
|------|------|
| リサイズ | 選択枠の8つのハンドルをドラッグ |
| 移動 | 移動ツール（V）でドラッグ |
| 切り取り後移動 | 選択後に V でドラッグすると内容を切り取って移動 |
| トリミング | 矩形選択後にツールバーの「⬚トリミング」ボタン |
| 自動選択オプション | 許容値・隣接のみ・アンチエイリアス・全レイヤー対象を設定可 |

## 保存形式

| 形式 | 透明度 | 用途 |
|------|:------:|------|
| PNG | ✅ | 透過画像・ロスレス |
| JPEG | ❌ | 写真・ファイルサイズ重視（クオリティ調整可） |
| BMP | ❌ | 非圧縮ビットマップ 24bit |
| TGA | ✅ | Targa形式 32bit |
| PSL | ✅ | PaintStudio独自形式・全レイヤー情報を保持 |

> PSL形式で保存すると、次回「画像読込」から同じファイルを開いてレイヤー構造を復元できます。

## 動作環境

Chrome / Edge / Firefox など、Canvas API と `ctx.filter` に対応したモダンブラウザ。  
タブレット・スタイラスのペン圧には非対応です。
