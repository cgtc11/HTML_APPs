# AEmini 機能説明書

AEminiはブラウザ（Chrome推奨）で動作する軽量タイムライン動画エディタです。  
After Effectsとの双方向JSON連携（Mini2AE.jsx）に対応しています。

---

## 素材パネル（左側）

### ファイルのドラッグ＆ドロップ
ブラウザ画面にファイルをドロップすると素材パネルに追加されます。

- 画像（PNG・JPG等）、動画（MP4等）、音声（MP3・WAV等）に対応
- 複数ファイルをまとめてドロップ可能
- 同じプレフィックス・同じ拡張子のグループは**連番素材（SEQ）**として自動認識

**空コンポへの1素材ドロップ時の特別動作：**
- コンポのサイズを素材の解像度に合わせて調整
- コンポの尺を素材の長さに合わせて調整（動画・音声・連番は実尺、静止画は10秒）
- 先頭レイヤーに自動配置
- タイムラインを素材の長さに合わせて自動フィット

### 素材の追加（ボタン）
| ボタン | 説明 |
|---|---|
| 単/連番素材を追加 | PNG・JPG等の静止画または連番画像を追加 |
| MP4動画を追加 | 動画ファイルを追加 |
| 音楽を追加 | MP3・WAV等の音声ファイルを追加 |
| 平面を作成 | 単色の平面素材を作成 |

### 素材の種類とバッジ
| バッジ | 種類 |
|---|---|
| SEQ | 連番画像 |
| IMG | 静止画 |
| MP4 | 動画 |
| AUDIO | 音声 |
| SOLID | 平面（単色） |
| TEXT | テキスト |
| COMP | コンポジション |
| ?? | リンク切れ（赤）→「リンク」ボタンで再指定 |

## プレビューエリア（中央上）

### 表示倍率
Fit / 25% / 50% / 100% ボタンで切り替え。

### プレビューでの操作
- クリップを選択した状態でドラッグ → **移動**
- 四隅のハンドルをドラッグ → **スケール変更**（X/Y個別）
- Shift+コーナードラッグ → **等倍スケール変更**
- プレビューとタイムラインの境界線をドラッグ → **エリアサイズ変更**

## タイムライン

### 起動時のデフォルト構成
新規コンポは **V2・V1・A1** の3段レイヤーで始まります。

### クリップの端をドラッグ
| 操作 | 動作 |
|---|---|
| 通常ドラッグ（静止画/平面） | 自由に伸縮 |
| 通常ドラッグ（SEQ/動画）右端 | 素材の尺まではトリミング、超えると最終フレームで静止（timeRemap自動作成） |
| Shift+ドラッグ | timeRemap KFで先頭・末尾を2点設定してスロー/ファスト伸縮 |

### ズーム
「ズーム」の文字をクリックするとコンポ全体がタイムライン幅にフィットします。  
簡易モード時はルーラー右端に「ズーム」ボタンが表示されます。

### クリップ操作
| 操作 | 説明 |
|---|---|
| クリップをクリック | 選択 → プロパティ表示 |
| クリップをドラッグ | 移動 |
| Shift+ドラッグ | スナップしながら移動 |
| Ctrl+C / Ctrl+V | コピー / プレイヘッド位置にペースト |
| Delete / Backspace | 削除 |
| Ctrl+Shift+D | プレイヘッド位置で分割 |

### マーカー
- **Mキー** または **◆Mボタン** → マーカーを追加（ラベル入力可）
- マーカーをクリック → その位置にジャンプ
- 右クリック → 削除

## プロパティパネル（右側）

プロパティは上から順に以下の順で並んでいます。

### ① 合成モード
通常・スクリーン・加算・乗算・オーバーレイ・反転（差分）から選択。

### ② テキスト内容（テキストクリップのみ）
テキスト本文・フォントサイズ・色・整列・フォント・太字/斜体・縁取りを変更。

### ③ 配置
| 項目 | 説明 |
|---|---|
| 長さ | クリップの表示時間 |

### ④ タイムリマップ（🟡 黄）
ソース時間をKFで自由にマッピングできます。スロー・ファスト・逆再生・ストップフレームに対応。

**KFの動作：**
- ◇ボタン → KFを有効化（先頭・末尾に2点KF自動作成）
- ◆ボタン → 現在時刻にKF追加
- -ボタン → 全KFを削除（無効化）

**ストップフレーム（■）の設定：**
- KFリストの◆をクリック → ■（ホールド）に切り替え
- タイムラインのダイヤをクリック（ドラッグしなければ） → ■に切り替え
- 再度クリックで◆に戻す

### ⑤ トランスフォーム（KF対応）

**ラベルドラッグで数値スクラブ（速度で変化量が変わる）**

| 項目 | タイムライン色 | 説明 |
|---|---|---|
| 位置 X/Y | 🔴 赤 | 水平・垂直位置。プレビューでもドラッグ可 |
| 回転 | 🔵 青 | 回転角度（°）。上下1°単位、入力は0.1°まで |
| 拡縮 X/Y | 🟢 緑 | X/Y個別スケール。「拡縮」ラベルドラッグでXY等倍 |
| 透明度 | 🟣 紫 | 不透明度（0〜1） |

**KFの操作（全プロパティ共通）：**
1. ◇ボタン → KF有効化（現在値で1点目を作成）
2. ◆ボタン → 現在時刻にKF追加
3. -ボタン → 全KF削除
4. KFリストの◆をクリック → ■（ホールド）に切り替え
5. タイムラインのダイヤをクリック → ■に切り替え

**KFデータ構造：**
- 位置: `{t, x, y, hold?}` 形式（X/Y共通KF）
- 拡縮: `{t, sx, sy, hold?}` 形式（X/Y共通KF）
- 回転/透明度/ぼかし/タイムリマップ: `{t, v, hold?}` 形式

### ⑥ 実サイズ
現在の拡縮を反映した実際のピクセルサイズを表示。

### ⑦ エフェクト
| 項目 | タイムライン色 | 説明 |
|---|---|---|
| ガウスぼかし | 🩵 水色 | ぼかし量（0〜100）、KF対応 |

### ⑧ クリップカラー
タイムライン上のクリップ表示色を変更。

### プロパティのリセット
移動・回転・拡縮・透明度・ぼかし・タイムリマップ・速度・合成モード・クリップカラーを初期値に戻し、全KFを削除します。素材の長さも元に戻ります。

## ペンツール

1. ✏ ペンボタン でON（プレビュー背景が赤くなる）
2. プレビュー上でドラッグして描画
3. **右ドラッグ** → 消しゴム
4. 描いた内容は「ペン Xf」素材として自動追加
5. 「PNG書き出し」ボタンで単独保存可能

> ⚠️ 通常保存ではペン素材はJSONに内包。「収集保存」または「PNG書き出し」で明示的に保存してください。

---

## テキストレイヤー

1. **T+ボタン** → 作成ダイアログ
2. テキスト内容・フォントサイズ・色・整列・フォント・太字/斜体・縁取りを設定
3. 「作成」→ 先頭レイヤーに5秒のクリップが配置される
4. KF（移動・回転・拡縮・透明度）も通常クリップと同様に設定可能

## 書き出し

### MP4書き出し
コンポのプロパティ → 「▤ MP4書き出し」ボタン  
ビットレート：4/8/16/30 Mbps。**Chrome 94以降 + WebCodecs対応が必要**

### 連番書き出し
コンポのプロパティ → 「▤ 連番書き出し」ボタン  
PNG・JPG・BMPをフォルダに出力。File System Access API対応ブラウザ必須。

## プロジェクトの保存と読み込み

### 📂 読込
JSONを読み込みます。素材が元の場所にあれば自動復元。  
見つからない素材は「⚠ リンク切れ」→「リンク」ボタンで再指定。

### 💾 保存
JSONファイルとして保存。通常の素材はパスを記録するだけ（ファイルはコピーされません）。

### 💾 収集保存
JSONと全素材ファイルをフォルダにまとめてコピー。別PCへの移動や整理に使用。

## After Effectsとの連携（Mini2AE.jsx v1.2）

AEのスクリプトメニューから `Mini2AE.jsx` を実行。

### Import（AEmini JSON → AE）
1. 「素材フォルダ」を指定
2. 「読み込み」ボタンでJSONを選択

**対応内容：**
- 映像/音声クリップの配置・タイミング
- 位置・回転・拡縮X/Y・透明度（静的値 + KFアニメーション）
- **ガウスぼかし（KFアニメーション含む）**
- **タイムリマップ（ストップフレーム/hold含む）**
- **holdフラグ → AEのHoldキーとして設定**
- テキストレイヤー（フォント・サイズ・色・縁取り）
- 平面レイヤー、フォルダ構造
- 素材が見つからない場合はグレー平面で代用（名前に[不明]付加）

### Export（AE → AEmini JSON）
1. AEでコンポを開いた状態でスクリプトを実行
2. 「対象」を選んで「書き出し」

**対応内容：**
- 位置・回転・拡縮（X/Y個別）・透明度（KFのhold含む）
- **ガウスぼかし（KFも書き出し）**
- **タイムリマップ（holdキーも書き出し）**
- テキストレイヤーの内容・書式
- フォルダ構造・平面レイヤー

---

## 注意事項・制限

- **Chrome推奨**（File System Access API使用のため）
- MP4書き出しは Chrome 94以降 / WebCodecs対応が必要
- 音声プレビューは再生時のみ（シーク精度に限界あり）
- 連番画像の最大フレーム数はメモリに依存
- ペン素材は通常保存でJSONに内包、ファイルとしては収集保存が必要
- ブラウザをリロードすると再生位置・選択状態はリセットされる
